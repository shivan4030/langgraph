When set to `True`, it will emit a `UserWarning` about the security risk. Update `loads_typed` to use `_RestrictedUnpickler(io.BytesIO(data_), allowed_modules=self._allowed_pickle_modules).load()` instead of `pickle.loads(data_)`.
